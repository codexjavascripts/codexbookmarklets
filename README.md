What is a Bookmarklet? > A bookmarklet is a normal browser bookmark that executes JavaScript instead of opening a web page.

🚀 How to Install
Make sure your browser's Bookmarks Bar is visible (Ctrl+Shift+B or Cmd+Shift+B).
Visit [https://codexjavascripts.github.io/codexgamingbookmarklets/).
Drag any button from the website directly onto your browser's bookmarks bar.
Navigate to any website you want to target, and click the bookmarklet to run it!
